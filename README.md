# Ycm自动化运维平台
* 此平台改写自oms，以及参考刘天斯自动化运维一书，目前还未完成

## 开发环境  
1.1 cestos 7  
1.2 python 2.75  
1.3 Django 1.8  

## 目录结构  
### Ycm
    settings.py 基本配置  
    settings_local.py 本地配置    

#### ycm 用户管理  
* asset 资产管理  
* audit 运维审计  
* server 服务配置  
* monitor 监控
* logserver 日志分析
